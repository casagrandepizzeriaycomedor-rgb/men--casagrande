# Casa Grande Pizzeria y Comedor - Menu Digital

Menu digital para Casa Grande Pizzeria y Comedor, ubicado en Corinto, Morazan, El Salvador.

## Que es

Una aplicacion web progresiva que permite a los clientes explorar el menu completo del restaurante desde su celular o cualquier dispositivo. Incluye:

- Categorias organizadas: Pizzas, Entradas, Comida, Pupusas, Antojitos, Sopas y Bebidas
- Buscador de platillos
- Modo oscuro / claro
- Informacion del negocio y redes sociales
- Boton de WhatsApp para pedidos directos
- Modal de detalle por platillo
- Imagenes de productos (pendiente: subir fotos reales)

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS v4
- GitHub Pages (hosting)

## Desarrollo local

```bash
npm install
npm run dev
```

## Build y deploy

```bash
npm run build
npx gh-pages -d dist
```
